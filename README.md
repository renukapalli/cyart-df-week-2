This repository covers:

Mobile Forensics (without tools)
Aviation Forensics
Digital Electronic Meter Forensics

1.Mobile Forensics (without tools)
Methodology:
Seizure → isolate device
Acquisition → duplicate data
Analysis → examine memory, SIM
Reporting → document findings
Techniques:
Manual inspection
Logical & physical extraction
Chip-off, micro-read
Use Cases:
Call/SMS recovery
Location tracking
Deleted data recovery

2. Aviation Forensics
Methodology:
Evidence collection from crash site
Wreckage documentation
Witness interviews
Flight data analysis
Techniques:
Structural & material analysis
Black box analysis
Crash reconstruction
Use Cases:
Accident investigation
Safety improvement
Failure analysis

3. Digital Electronic Meter Forensics
Methodology:
Analyze smart meter logs
Monitor network activity
Verify data authenticity
Techniques:
Log analysis
Network forensics
Pattern analysis
Use Cases:
Detect electricity theft
Identify cyber-attacks
Track user activity
